# 파이썬 머신러닝 완벽 가이드

머신러닝의 바이블 같은 책으로 양대산맥인 '핸즈온 머신러닝'(일명 도마뱀 책)과 비교해보며 고민 끝에 결정한 책입니다.

홀로 공부중이었으나, 마침 활동하고 있는 대학 연합 빅데이터 동아리에서의 교재로 선정되어 정리해가며 공부하려고 합니다.

이후 커리큘럼이 진행되면 수정/추가/보완할 예정입니다.
